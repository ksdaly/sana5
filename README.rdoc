== SANA

Sana provides online tools that help people lose weight and develop healthy, sustainable habits which will reduce their cardiovascular and diabetes risks.

== Setup

After checking out the application, copy `/config/database.example.yml` to `/config/database.yml`. These files contain sample database credentials required to run the application.

<b>After creating the database, run 'rake db:seed'. It is required to populate database with appropriate health plan and todo data.</b>
